## TO DO LIST 


# O QUE É ESTE PROJETO?

Este projeto denominado To Do List é um Gerenciador de tarefas no qual permite que qualquer pessoa adicione tarefas, as marque como finalizadas, delete-as ou as edite.

# Tecnologias
Frontend:
- React + Vite (JavaScript)

Backend:
- Node.js + Express



# COMO INSTALAR AS DEPENDÊNCIAS

Antes de iniciar o projeto, é necessário ter o **Node.js** instalado na máquina.

1. Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
```

2. Acesse a pasta do frontend:

```bash
cd To-do-List
```

3. Instale as dependências do frontend:

```bash
npm install
```

4. Acesse a pasta do backend:

```bash
cd backend
```

5. Instale as dependências do backend:

```bash
npm install
```



## Como executar o projeto localmente

### Executando o frontend

Na pasta principal do projeto:

```bash
npm run dev
```

O frontend estará disponível em:

```bash
http://localhost:5173
```



### Executando o backend

Na pasta `backend`:

```bash
node server.js
```

O backend estará disponível em:

```bash
http://localhost:3000
```

API de tarefas:

```bash
http://localhost:3000/tarefas
```
